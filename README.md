# Phicomm K2P OpenWrt Build

Minimal OpenWrt firmware build for Phicomm K2P router.

## Quick Start
1. Go to Actions tab
2. Run "Build Base Firmware"
3. Wait for completion
4. Download firmware

## Configuration
- Stage 1 (Base): Minimal system with LuCI
- Stage 2 (Plus): Added tools and features
- Stage 3 (Full): All plugins included

## Device Info
- CPU: MT7621AT
- RAM: 128MB
- Flash: 16MB
